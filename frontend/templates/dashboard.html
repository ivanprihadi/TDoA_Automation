  • Max 500MB per file
Recording Parameters:

Help & Documentation

User guide and FAQ

Getting Started

  1. Click "Start Recording" to begin a recording session
  2. Configure reference and target frequencies (FM band: 87-108 MHz)
  3. Set number of loops (each loop records ~3 seconds)
  4. Choose synchronization mode (NTP or GPS)
  5. Click "Start Recording" to initiate recording on all receivers
  6. Monitor progress in the progress dialog
  7. View results in the Results tab once complete

System Requirements

  • 3 synchronized RTL-SDR receivers
  • Known receiver coordinates (latitude, longitude, altitude)
  • Target frequency in FM band (87-108 MHz)
  • Reference frequency for calibration (must be different from target)
  • Network connectivity between PC and receivers

Troubleshooting

  • Recording stuck: Check if receivers are online and hardware connected
  • No results: Ensure sufficient signal strength at target frequency
  • Inaccurate location: Verify receiver coordinates and synchronization
  • Connection timeout: Check network connectivity and firewall settings
