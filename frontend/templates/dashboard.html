Getting Started

Welcome to TDOA Localization System! Here's how to get started:

  1. Prepare 3 IQ data files from your receivers
  2. Go to "Processing" tab
  3. Select the files and configure parameters
  4. Click "Start Processing"
  5. View results in "Results" tab

FAQ

Q: What file format is required?

A: IQ data files in .dat format from RTL-SDR devices

Q: How long does processing take?

A: Typically 30-60 seconds depending on file size

Q: Can I process multiple triplets?

A: Yes, use batch processing mode

System Requirements

  • CPU: 4 cores minimum
  • RAM: 8GB recommended
  • Storage: 50GB for signal data
  • Network: 3 Raspberry Pi receivers
  • Software: Python 3.8+
